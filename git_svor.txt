Git spurningar og svör

[1]

1. cd                         =  Er notað til að breytta um skrá sem þú ert í.
2. cd ~                       =  Er shortcut til að færa þig á fyrri skrá.
3. ls                         =  Býr til lista yfir skrám sem eru í skránni. 
4. pwd                        =  Er notað til að sjá í hvaða skrá þú ert í.
5. mkdir                      =  Er notað til að búa tilk nýja skrá.

[2]
   
1. git clone                  = Býr til eins geymslu í nýja skrá sem .þú ert í. 
2. git log                    = Sýnir öll commit sem þú hefur gert. 
3. got status                 = Sýnir status á working tree.
4. git diff                   = Sýnir breytingar milli commit og commit og working tree.
5. git checkout               = Endurheimtar tree skrár. 

[3]

a. git diff                   = Sýnir breytingar milli commit og commit og working tree.
b. git diff --staged          = Sýnir allar breytingar þú hefur gert.
c. git diff commit1 commit2   = Sýnir fyrstu tvær commits þú gerðir.
                                  
[4]                                
                                    
Kerfi sem skráir niður breyttingar sem þú hefur gert í skrám og leyfir þér að fara yfir 
breytingarnar                                    
                                    
[5]                                    
                                    
Þú getur alltaf farið á eldri punkt í verkefninu ,gert fleiri útgáfur af verkefninu
og gerir léttara að hoppa um útgáfur/verkefni.
                                     
[6]                                    
                                    
Þegar þú ert búinn að gera ákveðið mikið(fer eftir þér) og þegar þú hefur nýjar hugmyndir
sem þú ert ekki alveg viss um, þá getur farið alveg fullt í þessa hugmynd án þess að pæla
í miklu um hversu mikið þú þarft að breytta til að hafa allt eins og það var áður en þú 
komst með þessa hugmynd.                                    
                                     
[7]                                    
                                    
Woking directory              = Skráinn sem þú er í.
Staging area                  = Breyttar skrár sem eru tilbúnar fyrir næsta commit.
Repository                    = Þar sem þú geymir allt í þessum verkefnum í git.

[8]

þegar þú veist ekki hvernig verkefnið þitt á að vera þá geturu gert tvær útgáfur að því,
þesar útgáfur geta líka skipst og þú getur haldið áfram með hvaða útgáfu sem er.
